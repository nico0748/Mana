# src/components/books/

書籍管理機能の中核をなすコンポーネント群を格納するディレクトリ。書籍の一覧表示・詳細表示・追加・編集・削除・バーコードスキャンを担う。

## ファイル

### `BookList.tsx`

書籍管理のメインページコンポーネント。書籍の一覧表示と各種操作のエントリーポイント。

**機能:**
- `useBooks()`フックから書籍データとCRUD関数を取得
- テキスト検索（タイトル・著者名の前方一致）
- 書籍種別フィルタリング（全て / 商業本 / 同人誌）
- ソート（登録日時 / タイトル / 著者）
- 「本を追加」ボタンクリックで`BookForm`モーダルを表示
- 書籍カードクリックで`BookDetailModal`を表示
- Framer Motionによるリスト入退場アニメーション

**状態管理（useState）:**
| 状態 | 説明 |
|-----|------|
| `searchQuery` | 検索キーワード |
| `selectedType` | フィルタリングする書籍種別 |
| `sortOrder` | ソート順 |
| `isFormOpen` | BookFormモーダルの開閉状態 |
| `selectedBook` | 詳細表示する書籍（`null`のとき未選択） |

---

### `BookItem.tsx`

個別の書籍を表示するカードコンポーネント。

**Props:**
| プロパティ | 型 | 説明 |
|-----------|-----|------|
| `book` | `Book` | 表示する書籍データ |
| `onClick` | `() => void` | カードクリック時のコールバック（詳細モーダルを開く） |
| `onEdit` | `() => void` | 編集ボタンクリック時のコールバック |
| `onDelete` | `() => void` | 削除ボタンクリック時のコールバック |

**表示内容:**
- 書影画像（未設定時はプレースホルダー）
- タイトル・著者
- 書籍種別バッジ（商業 / 同人誌）
- 所有状態バッジ（所持中 / 貸出中 / 欲しい本）
- 編集・削除ボタン

**アニメーション:** Framer Motionの`layoutId`を使い`BookDetailModal`とShared Layout Animationを実現。

---

### `BookForm.tsx`

書籍の新規追加・編集を行うフォームコンポーネント。

**Props:**
| プロパティ | 型 | 説明 |
|-----------|-----|------|
| `initialData` | `Book \| null` | 編集時の初期値。`null`なら新規追加モード |
| `onSubmit` | `(data, file?) => void` | フォーム送信時のコールバック |
| `onCancel` | `() => void` | キャンセル時のコールバック |

**フォームフィールド:**
- タイトル（必須）
- 著者（必須）
- ISBN（任意）
- 書籍種別（商業 / 同人誌）
- カテゴリ（任意）
- 所有状態
- 書影（ファイルアップロード or URL入力）
- メモ（任意）

**追加機能:**
- ISBNフィールド横の「スキャン」ボタンで`BarcodeScanner`モーダルを起動
- ISBN入力後「取得」ボタンで`fetchBookByIsbn()`を呼び出しフォームを自動補完
- タイトル入力後「書影を検索」ボタンで`searchBookByTitle()`を呼び出し書影URLを自動設定

### `BookDetailModal.tsx`

書籍詳細を表示するモーダルコンポーネント。閲覧モードと編集モードを切り替えられる。

**Props:**
| プロパティ | 型 | 説明 |
|-----------|-----|------|
| `book` | `Book` | 表示する書籍データ |
| `onClose` | `() => void` | モーダルを閉じるコールバック |
| `onUpdate` | `(data, file?) => void` | 書籍更新時のコールバック |
| `onDelete` | `(bookId) => void` | 書籍削除時のコールバック |

**モード:**
- **閲覧モード**: 書籍の全フィールドを整形して表示。編集・削除ボタンあり
- **編集モード**: `BookForm`を`initialData`付きで表示。`isEditing`ステートで切り替え

**アニメーション:** Framer Motionの`layoutId`（`BookItem`と共有）によりカードからモーダルへのシームレスなアニメーションを実現。

---

### `BarcodeScanner.tsx`

カメラを使ってバーコード（ISBN）をスキャンするモーダルコンポーネント。

**Props:**
| プロパティ | 型 | 説明 |
|-----------|-----|------|
| `isOpen` | `boolean` | モーダルの表示状態 |
| `onScan` | `(isbn: string) => void` | スキャン成功時に呼ばれるコールバック |
| `onClose` | `() => void` | モーダルを閉じるコールバック |

**動作:**
1. `Modal`コンポーネント内にカメラ映像を表示
2. `react-zxing`の`useZxing()`フックでリアルタイムにバーコードを検出
3. ISBNが読み取れたら`onScan(isbn)`を呼び出してモーダルを閉じる
4. 閉じるとカメラストリームを停止

**依存:** `src/components/ui/Modal.tsx`, `react-zxing`
